# Sales Data Analytics Project

## Overview
Analyzed sales data to identify trends, key performance metrics, and business insights using Python and Power BI.

## Tools Used
Python (Pandas, NumPy, Matplotlib, Seaborn), SQL, Excel, Power BI, Google Colab

## Key Analysis
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- KPI calculation
- Interactive Power BI dashboard
  
## KPIs
- Total Revenue
- Monthly Sales Trend
- Top Products
- Region-wise Performance
- Average Order Value (AOV)
